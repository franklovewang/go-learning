go语言没有面向对象的概念，但是可以使用结构体来实现面向对象编程的一些特性，例如：继承，组合等特性。

### go语言结构体的定义

```
type struct_variable_type struct {
	member definition;
	member definition;
	...
	member definition;
}
```
type: 结构体定义关键字
struct_variable_type: 结构体类型名称
struct: 结构体定义关键字
member definition: 成员定义

**实例**

```
type Person struct {
	id int
	name string
	age int
}
```

**声明一个结构体变量**

声明一个结构体变量和声明一个普通变量相同，例如：

```
package main

import "fmt"

type Person struct {
    id   int
    age  int
    name string
}

func main() {
    var tom Person
    fmt.Printf("tom, %v\n", tom)
    kite := Person{}
    fmt.Printf("kite: %v\n", kite)
}
```

**访问结构体成员**

```
package main

import "fmt"

type Person struct {
    id   int
    age  int
    name string
}

func main() {
    var tom Person
    tom.id = 1
    tom.age = 20
    tom.name = "frank"
    fmt.Println(tom)
}
```

**匿名结构体**

如果结构体是临时使用的，可以不用起名字，直接使用，例如：

```
package main

import "fmt"

func main() {
    var dog struct {
        id   int
        name string
    }
    dog.id = 1
    dog.name = "xiaohei"
    fmt.Printf("dog: %v\n", dog)
}
```
